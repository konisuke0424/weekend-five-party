# インフルエンサーゲーム

スマホブラウザで遊ぶ5人向けカード対戦ゲームです。
iPhone/Androidはアプリ不要で、公開URLまたはQRコードから参加します。

## 現在の実装範囲

- ルーム作成/参加
- 最大6人参加
- ロール配布（効果あり）
- 初期フォロワー1000人
- v0.3カードプール（18種127枚）から80枚共有デッキをランダム生成
- 各プレイヤーに初期手札5枚
- 毎ターン開始時に1枚ドロー
- 手札カード使用、対象プレイヤー選択
- 引退、炎上、サブスク、アンチ、鍵垢などの状態管理
- デッキ切れまたは自分以外全員引退による勝敗判定
- 開発者テスト用CPUモード

## 実装済みカード（v0.3 / 18種）

- 毎日配信（サイコロ×1000、使っても手札に残る）
- 配信おやすみ（2枚ドロー）
- コラボ配信（対象と自分のフォロワー+10万）
- なりすまし（相手と同数、1/2で自分炎上）
- 重大なお知らせ（1/2で×1000、1/2で÷1000）
- 緊急で動画を回す（獲得+10%・永続重複可）
- 暴露（相手フォロワー半減）
- 炎上（相手を炎上状態）
- 謝罪（炎上回復+サイコロ×1000）
- 鍵垢にする（攻撃1回無効化・3ターン）
- くそマロ（相手手札3枚を置換）
- サブスク開始（毎ターン+10000）
- プレミアムサブスク（サブスク状態時に昇格・毎ターン+10万）
- 渾身のネタ（1/2で+100万人）
- 業界の闇暴露（全員50%減）
- アンチを送る（毎ターン-10000）
- 企業案件（+1万人）
- トレンド入り（獲得+30%・永続重複可）

## ローカル起動

```bash
npm install
npm run dev
```

開発時は `http://localhost:5173` を開きます。

本番相当で確認する場合:

```bash
npm run build
npm start
```

この場合は `http://localhost:3001` を開きます。

## クラウド配布

Render / Railway / Fly.io などのNode.jsサービスで動かせます。

- Build command: `npm install && npm run build`
- Start command: `npm start`
- Node version: `20` 以上
- Port: 環境変数 `PORT` が自動で渡されます

デプロイ後、公開URLをスマホで開いてルームを作成してください。
同じURL内にルームコード付きの招待リンクとQRコードが表示されます。

## 1人テスト

1. `Create Room` でルームを作成します。
2. `CPU Test` を押します。
3. 自分以外の4人がCPUとして追加され、5人対戦が始まります。
4. CPUのターンはサーバ側で自動進行します。
